## 修复 StatusFragment 闪退问题
- 原因：onCreateView 中立即调用 updateStatus 时，Fragment 可能还未完全 attach 到 Activity
- 解决方案：
  1. 将视图引用改为类成员变量
  2. 添加空值检查和 try-catch 保护
  3. 在 onViewCreated 中首次更新状态（确保视图已创建）
  4. 简化 updateStatus 方法参数，使用成员变量
